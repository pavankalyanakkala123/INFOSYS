=========================================
      INSTALLATION QUICK STEPS
=========================================

1. Install Python
   Download from: https://www.python.org/downloads/
   Add to PATH: ✓ CHECK THIS

2. Install Ollama
   Download from: https://ollama.ai/
   Pull model: ollama pull phi3:mini

3. Setup Project
   python -m venv .venv
   .venv\Scripts\Activate.ps1
   pip install -r requirements.txt

4. Run Application
   Terminal 1: ollama serve
   Terminal 2: streamlit run app6.py
   Browser: http://localhost:8501

TOTAL TIME: 15-30 minutes

=========================================
          TROUBLESHOOTING
=========================================

"Cannot connect to Ollama"
  → Run: ollama serve

"ModuleNotFoundError"
  → Activate venv: .venv\Scripts\Activate.ps1
  → Reinstall: pip install -r requirements.txt

"Ollama 500 Error"
  → Restart: Ctrl+C then ollama serve

"OCR not working"
  → Try: PNG format
  → Ensure: Readable text in image

"Port in use"
  → Run: streamlit run app6.py --client.serverPort=8502

See REQUIREMENTS.TXT for complete troubleshooting!
