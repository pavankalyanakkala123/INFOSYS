================================================================================
                    5-MINUTE QUICK START
================================================================================

STEP 1: Install Python
  1. Download Python from https://www.python.org/downloads/
  2. Run installer
  3. ✓ CHECK "Add Python to PATH"
  4. Verify: Open terminal → python --version

STEP 2: Install Ollama
  1. Download from https://ollama.ai/
  2. Run installer
  3. Open terminal → ollama pull phi3:mini
  4. Wait for download (~2.2GB, takes 5-10 minutes)

STEP 3: Create Virtual Environment
  Navigate to your project folder, then:
  
  Windows:
    python -m venv .venv
    .venv\Scripts\Activate.ps1
  
  Linux/macOS:
    python3 -m venv .venv
    source .venv/bin/activate

STEP 4: Install Dependencies
  pip install -r requirements.txt
  (Takes 5-10 minutes, first time only)

STEP 5: Run the App
  
  Terminal 1 - Start Ollama:
    ollama serve
    (Leave open)
  
  Terminal 2 - Start App:
    streamlit run app6.py
  
  Terminal opens browser automatically to:
    http://localhost:8501

DONE! ✓

================================================================================
                        TROUBLESHOOTING QUICK FIX
================================================================================

"Cannot connect to Ollama"
  → Run: ollama serve (in separate terminal)

"PaddleOCR not found"
  → Run: pip install paddleocr paddlepaddle

"Port 8501 already in use"
  → Run: streamlit run app6.py --logger.level=info --client.serverPort=8502

"Virtual environment won't activate"
  → Try: python -m venv .venv (recreate)

"pip install fails"
  → Run: pip install --upgrade pip setuptools wheel
  → Then: pip install -r requirements.txt
